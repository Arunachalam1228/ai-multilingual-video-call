# AI Multilingual Video Communication System

## Project Description
This project enables real-time video communication between users speaking different languages.
It uses AI to convert speech to text, translate it into another language, and convert it back to speech.

## Technologies Used
- Frontend: React
- Backend: Node.js + Express
- Video Call: WebRTC
- Speech to Text: AI Speech API
- Translation: AI Translation API
- Text to Speech: AI TTS API
- Database: MongoDB

## Features
- Real-time video calling
- Speech-to-text conversion
- Language translation
- Text-to-speech output
- User-friendly interface
